User Stories¶

Personas¶

Service Manager
person who is responsible for the performance of a service
External Developer
person who is responsible for integrating with the Performance Platform. This could be someone both inside and outside of government. The important aspect is that they are not a part of the Performance Platform development team

Stories¶

As a... I want to... So that I...
Service Manager [1] know what the Performance Platform is and why it’s important understand the benefits of integrating with the product
Service Manager [2] know how much work is involved in working with the Performance Platform can schedule any work needed
Service Manager [3] know what help the Performance Platform can offer my non-transactional service understand the benefits of integrating with the product
Service Manager [4] know the roadmap of features for the Performance Platform can know when to wait and when to roll my own
Service Manager [5] know how I can make updates to data when incorrect data is published have confidence that accurate figures are reported
Service Manager [6] know I can add editorial to my dashboard can explain interesting events or fluctuations
Service Manager [7] know how I can get a dashboard can start benefiting from the product
Service Manager [8] know what capabilities I need to do the integration work can know what I’m looking for when I need to procure external expertise
Service Manager [9] need to get the KPIs and dashboard onto the Performance Platform pass the service assessment
Service Manager [10] want the most accurate information presented in my KPIs can accurately determine how good (customer satisfaction) and how expensive (cost per transaction) my service is
Service Manager [11] want to be able to quickly communicate how different parts of my service is performing to the business don’t have to go to multiple data sources
Service Manager [12] want to be able to use Performance Platform data to help answer user needs on my service am confident the service is fit for purpose
Service Manager [13] want to be able to use Performance Platform data to recommend making changes in other parts of the service can improve the whole service, eg identify points of high friction in the user journey
Developer [14] know what data types I can put into the product know what is relevant to my service
Developer [15] know how to put data into the product can write suitable stories in my backlog and have sensible points for planning
Developer [16] know how to instrument my digital service am confident I am capturing the relevant data
Developer [17] know how to pull data out of the product can create custom visualisations or new insights
